# 基础类型

- 布尔值 `boolean`
- 数字 `number`
- 字符串 `string`
- 数组
    - 元素类型后面接上 [] `元素类型[]`
    - 数组泛型 `Array<元素类型>`
- 元组 Tuple `表示一个已知元素数量和类型的数组，各元素的类型不必相同。`
- 枚举 `enum`
- Any `any`
- Void `void`
- Null `null`
- Undefined `undefined`  
    默认情况下null和undefined是所有类型的子类型。
- Never `never`  
    表示的是那些永不存在的值的类型
- Object `object`  
    表示非原始类型
- 类型断言 `as`
